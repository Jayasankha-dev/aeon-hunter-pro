Privacy Policy

Last Deployment: February 5, 2026

Aeon Hunter Lite is a decentralized, local-first security auditing toolkit. Our architecture is designed to ensure that no user data ever leaves the local environment.

01. DATA USAGE

The extension monitors and processes specific web data exclusively for real-time visualization on your dashboard:

Network Monitoring Captures headers, methods, and endpoints for traffic analysis.
DOM Analysis Identifies input fields and API parameters for reconnaissance.

02. STORAGE & SECURITY

All intercepted data is stored in the browser's localSession or memory. We have zero backend infrastructure, meaning we have no physical way to collect or store your information.

03. THIRD-PARTY DISCLOSURE

We do not share, sell, or trade any data. This extension is a stand-alone utility for security professionals and does not include any third-party tracking scripts or telemetry.
